Les réputations de Cataclysme

Cataclysme comporte 3 factions exclusives à la Horde et 3 factions exclusives à l'Alliance. Il y a, en plus, 5 factions disponibles pour chacun des deux camps rivaux.

  • Horde
  • Le Cartel Baille-fond
  • Le Clan Gueule-de-Dragon
  • Le Poing de Hurlenfer
  • Alliance
  • Gilnéas
  • Le Clan Marteau-Hardi
  • Les Gardiens de Baradin
  • Neutre
  • Les Vengeurs d'Hyjal
  • Le Cercle terrestre
  • Les Gardiens d'Hyjal
  • Ramkahen
  • Therazane
