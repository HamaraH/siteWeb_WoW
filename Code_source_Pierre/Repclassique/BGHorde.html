Les Profanateurs

Les Profanateurs livrent une bataille sans merci a la Ligue Arathor dans le bassin d'Arathi, pour exploiter ses inestimables ressources. Pour gagner en réputation auprès de La Ligue Arathor, il faut se battre dans le champ de bataille du Bassin d'Arathi. Chaque fois que votre équipe obtient 100 ressources, votre réputation va augmenter de 50 points. Etant donné que pour gagner, il vous faut obtenir 1600 ressources, chaque victoire vous octroyera 800 points de réputation. Vous l'aurez compris, pour atteindre le niveau exalté pour cette réputation, vous battre avec ferveur. Une fois exalté, vous pourrez acheter le tabard des Profanateurs contre des marques d'honneur obtenues en champ de bataille.

Les Voltigeurs Chanteguerre

Les Voltigeurs Chanteguerre livrent une bataille sans relache aux Sentinelles dans les goulets de Chanteguerre, théatre de nombreux affrontements entre l'Alliance et la Horde. Pour gagner en réputation auprès des Voltigeurs Chanteguerre, il faut se battre dans le champ de bataille des goulets de Chanteguerre. Chaque fois que votre équipe capture le drapeau de l'équipe adverse, votre réputation va augmenter de 100 points. Si vous remportez une bataille dans les goulets de Chanteguerre, vous remportez 100 points supplémentaires. Vous l'aurez compris, pour atteindre le niveau exalté pour cette réputation, vous devrez vous lever tot. Une fois exalté, vous pourrez acheter le tabard des Sentilles d'Aile-argent contre des marques d'honneur obtenues en champ de bataille.

Le Clan Loup-de-givre

Le Clan Loup-de-givre livre une guerre totale a La garde Foudrepique dans la Vallée d'Alterac pour la dominer. Pour gagner en réputation auprès du Clan Loup-de-givre, il faut se battre dans le champ de bataille de la Vallée d'Alterac. Chaque fois que votre détruit un batiment, de l'équipe adverse, vous gagnez XXX points de réputations. Si vous remportez une bataille dans la Vallée d'Alterac, vous remportez XXX points supplémentaires. Vous l'aurez compris, pour atteindre le niveau exalté pour cette réputation, il vous faudra etre impitoyable. Une fois exalté, vous pourrez acheter le tabard du Clan Loup-de-givre contre des marques d'honneur obtenues en champ de bataille.
