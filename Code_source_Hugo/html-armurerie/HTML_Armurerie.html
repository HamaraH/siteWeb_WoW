Qu'est ce que l'armurerie ?

L'armurerie, c'est LE repère des amateurs de mode sur World of Warcraft !

Dans cette partie du site, nous nous consacrerons spécifiquement aux plus belles armures du jeu ! Celles-ci sont divisées en 2 catégories :

  • Les armures obtenues dans les modes de jeu Joueur contre Joueur (réparties sur plusieurs "saisons")
  • Les armures obtenues dans les différents raids du jeu (réparties sur plusieurs "tiers")
